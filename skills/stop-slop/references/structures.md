# Structural Anti-Patterns

## 1. Binary Contrasts

| Pattern | Example |
|---------|---------|
| "Not because X. Because Y." | "Not because they lacked talent. Because they lacked direction." |
| "[X] isn't the problem. [Y] is." | "Speed isn't the problem. Direction is." |
| "It's not about X. It's about Y." | "It's not about tools. It's about taste." |
| "The problem isn't X—it's Y." | "The problem isn't resources—it's priorities." |

**Fix:** State Y directly. The contrast adds nothing.

## 2. Negative Listing

Listing what something is NOT before saying what it IS.

| Pattern |
|---------|
| "It's not a framework. It's not a methodology. It's..." |
| "No dashboards. No standups. No retros. Just..." |

**Fix:** State the positive directly. Skip the runway.

## 3. Dramatic Fragmentation

| Pattern |
|---------|
| "[Noun]. That's it. That's the [thing]." |
| "One [noun]. [Short dramatic sentence]." |
| "[Statement]. Full stop." |

**Fix:** Complete sentences. Drama doesn't substitute for substance.

## 4. Rhetorical Setups

| Pattern |
|---------|
| "What if [reframe]?" |
| "Think about it:" |
| "Ask yourself:" |
| "Imagine [scenario]." |

**Fix:** Make the point directly. Don't stage it.

## 5. Formulaic Constructions

| Pattern |
|---------|
| "By the time X, I was Y." |
| "The moment I [verb], everything [verb]." |
| "[Time period] later, [revelation]." |

**Fix:** Vary the template. If it sounds like a writing exercise, restructure.

## 6. False Agency

Inanimate objects performing human actions.

| Pattern | Fix |
|---------|-----|
| "The complaint becomes a fix" | "Users who complain file patches" |
| "The decision emerges" | "The team decided" |
| "Quality finds a way" | "Engineers maintain quality by..." |
| "The code tells a story" | "Reading the code shows..." |

**Fix:** Name the human actor. Things don't decide, act, or feel.

## 7. Narrator-from-a-Distance

| Pattern | Fix |
|---------|-----|
| "Nobody designed this." | "You didn't plan for this." |
| "People tend to..." | "You probably..." |
| "Teams often discover..." | "You'll find..." |
| "Engineers sometimes forget..." | "You might forget..." |

**Fix:** Address the reader directly. "You" beats "People."

## 8. Passive Voice

| Pattern | Fix |
|---------|-----|
| "X was created" | "[Person] created X" |
| "Mistakes were made" | "[Team] made mistakes" |
| "The feature was shipped" | "We shipped the feature" |
| "It was decided that..." | "[Person] decided..." |

**Fix:** Find who did it. Make them the subject.

## 9. Sentence Starters to Avoid

| Starter | Why |
|---------|-----|
| Wh- openers (What, When, Where, Why) | Delays the point |
| "So" to start a paragraph | Conversational filler |
| "Look," or "Listen," | Performative authority |
| "And yet," | Manufactured tension |
| "But here's the thing:" | Throat-clearing |

**Fix:** Lead with subject and verb.

## 10. Rhythm Patterns

| Pattern | Fix |
|---------|-----|
| Three-item lists everywhere | Use two items. Or four. |
| Immediate question then answer | State the answer. |
| Punchy one-liner paragraph endings | Vary paragraph endings |
| Em-dashes for asides | Use commas or parentheses. Or restructure. |
| Staccato fragments for emphasis | Complete sentences. |

## 11. Word Patterns

| Pattern | Fix |
|---------|-----|
| Lazy extremes: "every," "always," "never" | Be specific about frequency/scope |
| All adverbs | Delete them. The verb should carry the meaning. |
